Mandelbrot Set Definition

A complex number c is a member of the Mandelbrot set if, when starting with $z_0 = 0$ and applying the iteration $z_{n+1} = z^2_{n} + c$ repeatedly, the absolute value of $z_{n}$ remains bounded; this point $c$ on the grid is then filled in.

Julia Set Definition

A complex number $z_0$ is a member of a Julia set if, when starting with $z_0$ and applying the iteration $z_{n+1} = z^2_{n} + c$ repeatedly, the absolute value of $z_{n}$ remains bounded. For the Julia set, $c$ is a fixed constant. If the sequence is bounded, with this value of $z_0$, the point $z_0$ is filled in on the grid.

Some choices of z0 which lead to interesting patterns are:

  • −0.5125 + 0.5213i
  • 0.37 + 0.16i
  • −0.50 − 0.56i
  • −0.25
  • i
  • −1.5
  • −0.75 + 0.25i
