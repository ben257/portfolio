About

Newton's Method is a root finding algorithm. To create the visualization, we take a polynomial function, defined by the inputted roots, and apply Newton's Method to each point on the grid, treating $(x,y)$ as $z=x+iy$. Points converging to the same root (i.e. points in the same basin) receive the same color.
