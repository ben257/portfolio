The L-System is a type of formal grammar. It consists of an alphabet of symbols used to form a string, an initial string (axiom), production rules which give the transformation of a symbol from one iteration to the next, , and a way to transform the string into a geometric structure.

Consider the following example. The alphabet is "-","+","F","G". The initial string is "F". The production rules are:
$\begin{align} F &\rightarrow F+G \\ G &\rightarrow F-G \\ + &\rightarrow + \\ - &\rightarrow - \end{align}$

These production rules would convert the string "FG-+" to "F+GF-G-+". To convert from the string to a geometric structure, the following rules are used. The initial position is at the origin, with an initial angle of 0. If a character is a forward character (F or G), a line is drawn oriented at the current angle, and the current position is now the end of the line. If the character is "+", the angle is rotated counterclockwise. If the character is "-", the angle is rotated clockwise.
